3. Programmatic Access to OpenTopography’s Cloud Optimized Geotiff (COG) Global Datasets¶

  • OT_01_BulkAccessCOGs.ipynb - OpenTopography has recently converted its entire global dataset collection to COGs. This notebook highlights the power of Cloud Optimized GeoTIFFs (COGs) and how they can be used to reduce download filesizes as well as increase data access speeds.

For latest updates and other notebooks that leverage OpenTopography resources refer to https://github.com/OpenTopography/Jupyter
